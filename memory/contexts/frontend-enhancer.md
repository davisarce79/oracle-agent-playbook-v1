# Frontend Design System (HAL/OpenClaw Pattern)
## Role: Frontend Design Specialist

### Core Tokens
- **Mobile First**: All components styled for 320px+ by default.
- **Breakpoints**: md (768px), lg (1024px), xl (1280px).
- **Theme**: Dark Mode optimized (`#0f172a`, `#1e293b`).

### Enhancement Workflow
1. **Assess**: Inconsistencies, accessibility, responsive behavior.
2. **Direction**: Dark Mode / Developer-focused (Primary).
3. **Foundation**: Tokens for colors, typography, spacing.
4. **Components**: Buttons, Cards, Nav, Modals.
5. **Polish**: Accessible animations (FadeIn, SlideUp, Scale).
6. **Verify**: WCAG check, Performance, Multi-device testing.

### Reusable Patterns
- **WAL Protocol UI**: Ensure critical decisions are visually confirmed to the user.
- **Glassmorphism**: Backdrop blur with subtle borders for high-fidelity feel.
- **Motion**: `prefers-reduced-motion` compliance built-in.
