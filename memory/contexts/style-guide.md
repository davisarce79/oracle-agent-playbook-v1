# Agent Gunna Design System & Principles (2026-03-09)

## Breakpoints
- **Mobile-First**: Default styles target small screens.
- **Desktop (lg)**: 1024px
- **Wide (xl)**: 1280px

## Core Principles
1. **Visual Hierarchy**: Leverage high-contrast typography (black/white/cyan) and generous spacing (bento grid) to guide user attention.
2. **Consistent Patterns**: Maintain the "Autonomous Operator" aesthetic (dark slate, neon accents, Lucide icons) across all touchpoints.
3. **Accessibility**: Always include ARIA labels and ensure keyboard navigation for all interactive elements (buttons, links).
4. **Performance**: Use CSS-only animations and optimized Tailwind transitions to keep load times low on OCI.
5. **Mobile-First**: Designs must be functional and beautiful on handheld devices before scaling up.
